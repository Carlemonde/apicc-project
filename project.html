The SOFIS Solution

Smart Orchard Ferti-Irrigation System (SOFIS) is the core system being developed in this project. It is based on the acquisition and processing of evapotranspiration (ET) data using low-cost devices and machine learning algorithms.

Integrated sensor array for Soil, Plant, and Atmospheric monitoring.
layers

Multi-Layer Sensing

Integrated monitoring of soil moisture, sap flow (plant status), and atmospheric demand (VPD, temperature, humidity).

psychology

AI-Driven Algorithms

Advanced Artificial Neural Networks (ANN) process raw sensor data to predict precise irrigation needs.

System Workflow

1

Hardware: LILYGO devices collect local sensor data at the orchard edge.

2

Transmission: Encrypted data sent over 4G/LTE to our cloud infrastructure.

3

Analysis: ThingSpeak and MATLAB engines run the SOFIS algorithm.

4

Action: End users receive visualized data insights and use them to support decision-making.
